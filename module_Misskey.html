  • visibleUserIds (list) – When “specified” is specified, if the user ID is put in this argument, the user can view the post.

  • viaMobile (bool) – Mark it as posted as mobile if you set it to True.

  • localOnly (bool) – You can specify whether to post only locally.

  • noExtractMentions (bool) – Specify whether to not expand mentions from the text.

  • noExtractHashtags (bool) – Specifies whether to not expand the hash tag from the text.

  • noExtractEmojis (bool) – Specify whether to not extract custom pictograms from the text.

  • fileIds (list) – You can specify the file ID attached to the post.

  • replyId (str or None) – Specify the noteID to reply.

  • renoteId (str or None) – Specify the note ID to be Renote.

  • poll (list) – Designate when voting. You can specify 2 to 10 items.

  • pollMultiple (bool) – Specifies whether to allow multiple votes.

  • pollExpiresAt (str or None) – Specifies the time to expire. If pollExpiresAt and pollExpiresAfter are not specified, it will be a vote indefinitely.

  • pollExpiredAfter (str or None) – Specify the specified period from the post. If pollExpiresAt and pollExpiresAfter are not specified, it will be a vote indefinitely.
